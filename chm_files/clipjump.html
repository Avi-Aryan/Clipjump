Clipjump

A Magical Clipboard Manager

version 7.8

Recommended Online Manual

A Multiple-Clipboard management utility for Windows.
Allows you to simultaneously use multiple clipboards like never before. Everything that is transferred to your clipboard will get automatically transferred to the Multiple-Clipboards.
It's fast , it's easy, it's magic.

Features

  • A True Clipboard Manager - Captures everything
  • Supports preview of Clipboard stored images such as Print-screen captures.
  • Data-Types are meaningless for Clipjump.It just stores everything.
  • Control Clipboards only by using Ctrl+x,Ctrl+c and Ctrl+V.
  • Not Mandatory to perform Copy and Cut operations using Ctrl+C and Ctrl+X. Even an application's Copy and Cut will be transferred to Clipjump.
  • Clipjump History Tool - View, search and restore old clipboard items.
  • FIX Clipboards at certain positions via FIXATE option.(See FIXATE - A Coders Delight)
  • Add selective data to System clipboard if needed.
  • Clipjump Channels - Have as many as multiple Clipboard channels in Clipjump that store data independent of each other.
  • Can have as much simultaneous clipboards as possible. (Depends on disk space).
  • Control Clipjump totally from external program(s) via Clipjump Controller
  • Very fast Clipboard switching.
  • Copy current folder path to Clipboard (Ctrl+Alt+X)
  • Copy selected files path(s) to Clipboard (Ctrl+Alt+C)
  • Copy selected file content to Clipboard (Ctrl+Alt+F)
  • Idiot-Proof! - Captures Clipboard entries from everywhere, whether it be an external program or Print-screen captures.
  • No-Formatting - Remove formatting from clipboard clips on the fly
  • Unicode - Supports all characters
  • Incognito Mode - Disable History when needed
  • Easily Configurable - You can change settings via a GUI.
  • Not touches the registry and System drive
  • Free, Portable, Open Source
